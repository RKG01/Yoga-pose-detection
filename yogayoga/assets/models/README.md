Place the MoveNet TFLite model here.

Copy the model file `movenet_thunder.tflite` from the repository root `classification model/` into this folder:

cp "../classification model/movenet_thunder.tflite" assets/models/movenet_thunder.tflite

Alternatively download the model from TF Hub and place it here.

Note: Large binary files may not be committed to some Git hosts; keep this in mind when sharing the repo.
